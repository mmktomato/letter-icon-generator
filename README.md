# letter-icon-generator

**EXPERIMENTAL IMPLEMENTATION**

Generates a svg or png image. A single letter is on center of the image.

![Sample](/sample.png)

## Prerequisite

* Node.js (>= 7.6.0)

## Development

1. Run `PUPPETEER_SKIP_CHROMIUM_DOWNLOAD=true npm install`
2. Run `npm start`
3. Open `http://localhost:1337`
